# Onboarding Process

Enterprise onboarding usually lasts 2 to 4 weeks depending on data migration and stakeholder availability.

New Enterprise customers receive:
- a kickoff session
- configuration guidance
- access setup support
- training materials
- weekly check-ins during rollout
